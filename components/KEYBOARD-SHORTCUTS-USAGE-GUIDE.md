# Keyboard Shortcuts Usage Guide

## Quick Reference

### Available Shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `N` | Create new ticket | Anywhere on tickets page |
| `/` | Focus search input | Anywhere (even in input fields) |
| `1` | Filter by Low priority | Tickets list |
| `2` | Filter by Medium priority | Tickets list |
| `3` | Filter by High priority | Tickets list |
| `4` | Filter by Urgent priority | Tickets list |
| `5` | Clear priority filter | Tickets list |
| `?` | Show keyboard shortcuts help | Anywhere |
| `Escape` | Close dialogs | When dialog is open |

## How to Use

### 1. Creating a New Ticket
1. Navigate to the tickets page
2. Press `N` key
3. The "Create New Ticket" dialog will open
4. Fill in the form
5. Press `Escape` to cancel or click "Create Ticket" to submit

### 2. Searching for Tickets
1. From anywhere on the tickets page, press `/`
2. The search input will be focused and selected
3. Start typing to search
4. Results will update automatically

### 3. Filtering by Priority
1. View the tickets list
2. Press number keys `1-5`:
   - `1` = Low priority
   - `2` = Medium priority
   - `3` = High priority
   - `4` = Urgent priority
   - `5` = Clear filter (show all)
3. The list will update immediately

### 4. Viewing All Shortcuts
1. Press `?` (Shift + /)
2. A help dialog will appear showing all available shortcuts
3. Shortcuts are grouped by category
4. Press `Escape` to close the help dialog

## Tips & Tricks

### Smart Input Detection
- Most shortcuts are disabled when typing in input fields
- This prevents accidental actions while entering text
- Exception: `/` and `Escape` work everywhere

## Troubleshooting

### Shortcut Not Working?
1. Check if you're in an input field (most shortcuts disabled)
2. Verify the correct key is pressed
3. Check if a dialog is open (some shortcuts context-specific)
4. Press `?` to see all available shortcuts

### Conflicts with Browser Shortcuts?
- The system prevents default browser behavior
- If a conflict occurs, the app shortcut takes precedence
- Common browser shortcuts (Ctrl+T, Ctrl+W) still work

### Platform-Specific Issues?
- Mac users: Use `Cmd` instead of `Ctrl`
- Windows/Linux users: Use `Ctrl`
- The help dialog shows the correct key for your platform

## Best Practices

### Learning Shortcuts
1. Start with the most common: `N`, `/`, `?`
2. Use the help dialog (`?`) as a reference
3. Practice one shortcut at a time
4. Build muscle memory through repetition

### Efficiency Tips
1. Use `/` to quickly search without clicking
2. Use number keys to filter while browsing
3. Use `N` to create tickets without scrolling
4. Use `?` when you forget a shortcut

### Combining Actions
1. Search with `/`
2. Filter with `1-5`
3. Create ticket with `N`
4. Navigate with `Tab` and `Enter`
